| **M3** 设备 + 课程 + 通知 | Day 6-7  | 13     | 设备 CRUD、借用申请/审批/归还、课程安排、通知发布         | 设备借用和课程排课可走通       |

**时间估算说明：**

每个任务给出三个估值（单位：小时）：

- **O** = 乐观 (Optimistic) — 一切顺利
- **P** = 悲观 (Pessimistic) — 遇到阻塞
- **ML** = 最可能 (Most Likely) — 正常节奏

期望工期 = (O + 4×ML + P) / 6（PERT 加权）。
