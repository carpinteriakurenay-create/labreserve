## 视觉风格总结

```
整体风格：
- 后台管理系统风格
- 白色内容卡片
- 浅灰页面背景
- 蓝绿色作为主色
- 表格线条浅色
- 按钮圆角较小
- 信息密度较高

主要组件：
- 左侧导航菜单
- 顶部面包屑
- 查询筛选栏
- 操作按钮组
- 数据表格
- 分页器
- 表单页
- 弹窗表单
- 数据统计卡片
- 折线图
- 柱状图
- 时间线消息
```
